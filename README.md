# Contact Management App

A simple Contact Management web application built using the MERN stack.  
This project demonstrates core full-stack development concepts including REST APIs, MongoDB schema design, React state management, and clean UI implementation.

## Features

Add a new contact (Name, Email, Phone, Message)
Client-side form validation
Disable submit button for invalid input
View saved contacts instantly (no page reload)
Delete contacts
Responsive and clean UI

## Tech Stack

### Frontend

- React.js (Vite)
- JavaScript
- CSS (custom styling)
- React Hooks (`useState`, `useEffect`)

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
